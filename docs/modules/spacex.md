# SpaceX

Provides details about the next SpaceX launch.

## Configuration

```yaml
spacex:
  enabled: true
  colors:
    label: "green"
    text: "white"
  position:
    top: 1
    left: 2
    height: 1
    width: 2
  refreshInterval: 3600
```
